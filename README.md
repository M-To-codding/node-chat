# node-chat

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Structure of project

- src (Vue application)
- config (Env vars, passport)
- controllers
    - models (MongoDB)
        - Chat.js
        - Room.js
    - routes
        - chat.js
        - room.js
- public (static files)
    - index.html
- app.js (main file)
